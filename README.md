# Octave

A modern playlist curator with fine-tuned control over playlist creation.

> *Find your flow.*

Octave lets users build playlists from seed tracks or playlists, visualize their music through interactive genre/mood insights, refine using audio feature sliders, and export directly to Spotify.

## Tech Stack

- **Frontend**: Electron + React + TypeScript + TailwindCSS + Framer Motion + D3.js
- **Backend**: Python + FastAPI + Spotipy
- **Storage**: SQLite (local)
